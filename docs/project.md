# Описание проекта «Домашняя автоматизация IoT»

## Общая информация
**Название проекта:** Домашняя автоматизация IoT (Разработка сервера и интеграция функций умного дома)  
**Автор:** Жуков Станислав Романович  
**Тип проекта:** Технический, инфраструктурный

## Актуальность
Большинство коммерческих решений умного дома зависят от зарубежных облачных провайдеров, что создаёт риски отключения сервисов или утечки персональных данных. Локальный IoT-сервер решает проблему конфиденциальности, работает без интернета внутри домашней сети и позволяет гибко масштабировать систему под любые нужды без абонентской платы.

## Цель проекта
Создать энергоэффективную, независимую от облачных зарубежных сервисов систему автоматизации жилья с гибким локальным сервером и возможностью удалённого управления для повышения комфорта, энергоэффективности и безопасности жилья.

## Задачи
- Изучить протоколы передачи данных (MQTT, HTTP)
- Спроектировать схему подключения периферийных устройств
- Развернуть серверную часть
- Протестировать отказоустойчивость системы при потере сети
- Создать веб-интерфейс управления (дашборд)
- Настроить автоматические сценарии работы устройств

## Техническая часть (выполненная в рамках практики)
- Развёрнут MQTT-брокер для обмена сообщениями между устройствами
- Установлена и настроена СУБД PostgreSQL
- Подключены физические датчики температуры и влажности
- Написана первая стабильная версия прошивки для контроллеров
- Созданы простые автоматические правила (движение → свет, температура → уведомление)

## Технологический стек
- MQTT, HTTP
- PostgreSQL
- HTML5, CSS3, JavaScript
- Git + GitHub
- Python (для серверной логики)

## Результаты
- Рабочая схема из сервера и двух подключённых датчиков
- Базовая автоматизация с текстовыми уведомлениями при изменении температуры
- Полноценный сайт практики
- Полная документация проекта
